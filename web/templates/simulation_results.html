This page should be used to display the visualization after the simulations. It will be embedded into the /simulate page and display after the user runs the Monte Carlo simulation with the parameters they specified in /simulate. The workflow goes as follows: index.html -> /simulate. Visualization, which is separate but embedded HTML file, is not visible. User enters parameters for up to 3 races for a selected season (year). User clicks "Run" -> editable parameters disappear. Loading circle appears. API request to run simulation with specified parameters. API responds with data for visualization (preferrably in JSON format). Loading circle disappers. Data is loaded into simulate_results.html from main.py. Then, main.py tells /simulate that simulation has finished running and was sent. /simulate unhides the embedded visualization page. Reset button unhides. User clicks reset button to hide visualization and show editable parameters.
